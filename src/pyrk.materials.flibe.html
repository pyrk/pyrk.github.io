pyrk.materials.flibe module¶

class pyrk.materials.flibe.Flibe(name='flibe')[source]¶

Bases: pyrk.materials.material.Material

This class represents FLiBe. It inherits from the material class and possesses attributes intrinsic to flibe.

density()[source]¶

FLiBe density as a funciton of T. [kg/m^3]

The relation is

2415 + 0.49072T

This is based on http://aries.ucsd.edu/raffray/publications/FST/TOFE_15_Zaghloul.pdf it is valid between the melting point and the critical point

t_m = 732.2

t_c = 4498.8
